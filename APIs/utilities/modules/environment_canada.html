static retrieveHourlyForecast(cityOrUrl, hourCount=12)[source]¶

Retrieves the hourly forecast for the given city.

Parameters:
  • cityOrUrl (str) – the city name or the Environment Canada website ‘https://www.weather.gc.ca/forecast/hourly/on-118_metric_e.html’
  • hourCount (int) – the # of forecast hour to get, starting from the next hour relative to the current time.
Return type:

list(Forecast)
Raise:

ValueError if cityOrUrl points to an invalid city, or if hourCount is more than 24 and less than 1.
