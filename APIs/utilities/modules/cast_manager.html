aaa_modules.cast_manager.playMessage(message, casts=[<aaa_modules.chromecast.ChromeCast instance>, <aaa_modules.chromecast.ChromeCast instance>, <aaa_modules.chromecast.ChromeCast instance>], volume=50)[source]¶

Play the given message on one or more ChromeCast and wait till it finishes (up to MAX_SAY_WAIT_TIME_IN_SECONDS seconds). Afterward, pause the player. After this call, cast.isActive() will return False.

If _testMode is True, no message will be sent to the cast.

Parameters:
  • message (str) – the message to tts
  • casts (list(ChromeCast)) –
  • volume (int) – the volume value, 0 to 100 inclusive
Returns:

boolean True if success; False if stream name is invalid.
Raise:

ValueError if volume is not in the 0 - 100 inclusive range, or if message is None or empty.
